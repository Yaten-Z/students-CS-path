## 0. 开始之前

作为一名初中生，我发现计算机科学其实比大多数人想象的更容易接触——只要你真正感兴趣且愿意学习。你不必是天才，只需怀揣好奇心、保持坚持不懈的精神并付出努力即可。

在开始学习前，你可能会有几个疑问：

1. **需要高性能电脑吗？**
   ***不需要。** 在入门阶段普通电脑完全够用。*
1. **需要先学数学吗？**
   ***不需要。** 数学决定你的上限而非下限。*
1. **英语很重要吗？**
***很重要。** 许多优质学习资源仅提供英文版本。*
1. **需要预先理解计算机工作原理吗？**
***不需要。** 学习过程中你自然会掌握这些知识。*
1. **需要整天钻研计算机科学吗？**
***不需要。**事实上，许多优秀开发者都保持着运动、阅读、旅行等爱好，~~以及变装~~。

现在，让我们开始这段计算机科学之旅。

当前阶段无需构建复杂项目，理解计算机原理对你更有价值。编程时需要感知计算机的运作逻辑。首先，你需要选择第一门编程语言。

## 1. 第一门编程语言

作为初学者，你肯定是无法立即掌握多种语言。精通一门核心语言对打牢基础更为重要。

### Python

Python 是入门门槛极低的语言。从人工智能到游戏开发，它无所不能。其代码可读性强且简洁，如同阅读英语般流畅。更拥有海量的第三方库（预编写代码）供你调用。

### C++

C++ 是国际信息学奥林匹克竞赛（OI）中最核心的编程语言，在 NOI（中国全国信息学奥林匹克竞赛）、USACO（美国青少年编程竞赛）等国内外重要赛事中均被允许甚至推荐使用。作为运行效率最高的语言之一，C++ 广泛应用于游戏引擎和高端软件开发领域，同时也能帮助学习者深入理解计算机的底层运行机制。

### Scratch

Scratch 是面向低龄学习者或零基础编程初学者的一种入门式编程工具。它通过图形化积木模块的方式代替传统的代码编写，使学习者能够以直观的方式理解程序的基本逻辑。然而，Scratch 更适合作为编程学习的起点，不宜长期停留。在掌握基本概念并具备一定编程思维后，学习者应逐步过渡到基于文本的编程语言，以进一步提升编程能力。

## 2. 学生核心技能


### Linux

学习使用 Linux 是程序员成长过程中的一个重要里程碑。世界上绝大多数服务器、超级计算机以及 Android 设备都运行在 Linux 系统之上。与 Windows 相比，Linux 提供了更加贴近底层、也更具本质性的编程环境。（事实上，MacOS 也是一个不错的选择，但对大多数学生而言，Mac的价格还是比较高。）

你不需要卸载现有的 Windows 或 macOS 系统。可以通过虚拟机（VM）运行 Linux，或使用 WSL（Windows 的 Linux 子系统）进行体验。在入门阶段，只需重点掌握一些基本的终端命令即可，例如 `ls`、`cd`、`mkdir`、`grep` 等。

### Git 与 GitHub

Git 是一种版本控制系统，用于记录代码的修改历史，并帮助开发者进行协作开发。GitHub 是一个用于托管 Git 仓库的平台，使开发者能够将代码发布到互联网上，并与他人共享。学习 Git 与 GitHub 可以帮助你更高效地管理项目，并与其他开发者开展协作。

在学习初期，可以从一些基础命令入手，例如 `git init`、`git add`、`git commit` 和 `git push`。随着熟练度的提升，可以进一步学习更高级的功能，如分支管理（branching）和拉取请求（pull request）等协作机制。

### Markdown

Markdown 是一种用于格式化文本的轻量级标记语言，被广泛应用于 GitHub、技术博客以及各类文档编写场景中。掌握 Markdown 有助于你编写更加清晰、规范的文档，并与其他开发者进行高效沟通。Markdown 的学习成本较低，通常只需几分钟即可入门，但这一技能将在整个编程学习过程中持续发挥作用。

### 信息检索（Searching）

具备良好的信息检索能力是每一位程序员都应掌握的重要技能。当你在学习或开发过程中遇到问题时，首先应尝试通过互联网进行搜索。应使用与问题高度相关的关键词，并耐心查阅论坛、官方文档以及各类技术教程。快速获取并筛选有效信息的能力，可以显著减少时间成本以及学习挫败感。

此外，合理使用 AI 工具（如 DeepSeek、ChatGPT 等）也有助于快速获取思路与解释，但需要注意对 AI 提供的信息进行验证，以确保其准确性和可靠性。

### 数学（Math）

虽然开始学习编程并不要求具备深厚的数学功底，但良好的数学基础往往会对理解编程概念有很大帮助。代数、逻辑以及离散数学等内容在编程学习中经常会被用到。随着学习的深入，你会逐渐接触到算法、数据结构等计算机科学领域知识，而这些内容通常需要更扎实的数学支撑。不过，这些知识可以在实践中逐步学习和掌握，因此不必在开始编程之前就完全精通相关数学内容。

### 英语（English）

英语是编程领域的通用语言。绝大多数编程语言采用英语作为关键字，相关的文档、教程以及技术论坛也主要以英语呈现。虽然在英语能力有限的情况下依然可以开始学习编程，但提升英语水平将帮助你接触到更丰富的学习资源，并拓展在程序员社区中的发展空间。可以从常见的技术词汇入手，在学习编程的过程中逐步提高阅读与写作能力。

### 搜索引擎与人工智能（Google and AI）

熟练使用搜索引擎和人工智能工具是程序员应具备的一项重要能力。当遇到问题或需要学习新知识时，通常可以通过网络搜索作为第一步。应合理选择与问题高度相关的关键词，并积极查阅论坛讨论、官方文档以及教学资料。

AI（如 DeepSeek、ChatGPT 等）同样能够在获取思路和解释方面提供帮助，但需要注意的是，AI 生成的内容可能存在不准确或不完整的情况，因此应通过其他可靠来源进行验证与补充。

## 3. 尽早构建项目

### 博客（Blog）

博客（Blog）是分享知识、记录学习历程的绝佳方式。它既能提升写作能力，又能帮助你在编程社区建立个人影响力。你可以通过GitHub Pages、Medium或WordPress等平台创建博客，记录学习心得、项目进展或任何有趣的编程话题。

博客还能成为展示个人项目的舞台，让你与他人分享技术洞见。通过博客建立联系，你将逐步拓展编程圈的人脉网络。

### 小型游戏开发

开发小型游戏是实践编程技能的趣味途径，既能享受创作乐趣又能巩固编程能力。可从井字棋、贪吃蛇或迷宫等简单游戏起步，随着经验积累逐步挑战更复杂的游戏。游戏开发还能帮助你掌握面向对象编程、事件处理和图形编程等重要概念。

### 个人主页（Personal Homepage）

个人网站（Personal Homepage）是建立网络形象、展示技能与项目的绝佳方式。运用HTML、CSS和JavaScript等前后端技术，可打造简洁的网站用于自我介绍、作品展示及联系方式呈现。这在未来申请实习或工作时尤为重要，它为他人提供了了解你和查看你的作品的平台。

你可以在GitHub Pages等平台免费托管个人网站，使其成为你的网络核心枢纽。将博客、GitHub主页及其他相关网络资源与之关联，形成完整的数字足迹。
